# PlataformasHTML
Desarrollo de paginas web simples en HTML:
CatBlog - Events: He desarrollado sitios web como una Single Page Website en HTML, utilizando un sistema de navegación por anclas internas que permite desplazarse por las distintas secciones sin salir de la misma página. De esta forma, el menú redirige directamente a los contenedores específicos mediante sus identificadores, logrando una navegación fluida, rápida y totalmente integrada en una sola interfaz. A su vez he incluido imagenes para hacerlas más visuales.
Travels: He desarrollado mi sitio web en HTML como un directorio centralizado donde el menú principal actúa como un puente hacia recursos externos. Cada enlace de la navegación está configurado para redigir a sitios externos o plataformas independientes, permitiendo que el usuario acceda a información alojada fuera de mi dominio de forma organizada y directa desde una interfaz sencilla.
Video: He disarrollado una página web sencilla en HTML centrada exclusivamente en contenido audiovisual, donde el menú principal sirve para organizar y redirigir a los usuarios hacia diferentes videos externos. La estructura está pensada para ser minimalista y directa, permitiendo que la navegación funcione como un selector que facilita el acceso rápido a cada pieza de video sin distracciones visuales.
FeedbackForm - Form: He desarrollado una página web sencilla en HTML cuyo propósito principal es funcionar como un formulario interactivo, integrando diversos métodos de captura de datos en una sola interfaz. A través del menú, el usuario puede navegar entre secciones diseñadas para la entrada de texto, menús de selección desplegable y opciones de selección múltiple, permitiendo recopilar y procesar la información de manera estructurada y eficiente según las necesidades de cada campo
